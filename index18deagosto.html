PASOS PARA CREAR UNA CUENTA EN GITHUB

Síguelos en orden

1.- Buscar Github en tu navegador o accede con este link Github.


2.- Dar clic en Sign in para llenar todos los datos que nos pide.


3.- Una vez colocado los datos, seleccionaremos nuestras preferencias para la cuenta.


4.- Al dar al completar, tendremos que confirmar el correo que nos enviaron.


5.- Cuando hemos confirmado el correo, ya tendremos las opciones para crear resposiorios.


6.- Al terminar de poner el nombre a nuestro respositorio le damos en Crear Repositorio.


7.- Daremos clic en Agregar archivo y luego en Subir archivo. Esto la haremos para que nuestra página sea subida a internet por medio de Github.
